# TP Spring RestTemplate

Ce dépôt contient deux projets :
- **Backend** : REST API utilisant Spring Boot + JPA + H2 (port 9090)
- **Frontend** : Application MVC JSP utilisant RestTemplate (port 9191)

## Lancement
1. Démarrer `resttemplate-back`
   - Accès console H2 :
     - http://localhost:9090/h2-console

2. Démarrer `resttemplate-front`
   - Interface :
     - http://localhost:9191/

## Fonctionnalités réalisées
✔ CRUD complet  
✔ Communication via RestTemplate  
✔ JSP + JSTL
